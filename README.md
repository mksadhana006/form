## Login Form Project

### Description

This project is a simple login form created using HTML and CSS.
It focuses on user interface design and basic form structure.
No backend authentication or database is included.

### Features

* Username and password input fields
* Password masking
* Required field validation using HTML attributes
* Simple and clean layout
* Responsive form design


### Technologies Used

* HTML5
* CSS3
* javascript


### Validation Details

* All fields are mandatory
* Password field uses secure input type
* Browser-based validation is applied

### Limitations

* No backend logic
* No database connection
* No real user authentication
